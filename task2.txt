1. Amazon Web Service - BigData және әртүрлі форматтағы файлдарды сақтауға арналған бұлттық жүйе десекте болады, оны бизнес аналитика жасауға және көптеген есептеулер жүргізуге көмектеседі.
2. Облачные сервисы (Cloud Computing) — бұл интернет арқылы компьютерлік ресурстарға қол жеткізу.
3. IaaS (Infrastructure as a Service) - Пайдаланушыға серверлер, сақтау орындары, желі сияқты инфрақұрылым беріледі.
PaaS (Platform as a Service) - Пайдаланушыға дайын платформа және құралдар беріледі, код жазып іске қосуға болады.
SaaS (Software as a Service) - Пайдаланушы интернет арқылы дайын бағдарламалық өнімді қолданады.
4. Amazon S3 - бұл Amazon Web Service ішіндегі бір сервис деуге болады, оның өзінің атқаратын қызметтері бар.
5. Bucket - Minio ішіндегі контейнер десекте болады, оған белгілі бір ережелер беріп қоюға болады, мысалы: private немесе public етуге болады, сонымен бірге bucket ішіне файлдарды папкаға бөліп салуға болады.
6. Object — бұл bucket ішіндегі нақты файл немесе деректер. Object — файл + метадеректер (файл туралы ақпарат: аты, форматы, өлшемі, т.б.)
